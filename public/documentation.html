DojoDB - Documentation

This web application allows users to explore information on movies with a special focus on martial arts films. Users can sign up, update their personal profiles, and curate a list of their favorite movies.

{
                                id: 1,
                                title: 'Drunken Master (1978)',
                                description: 'Jackie Chan stars as a young man who [...]',

Get data about a genre by name /movies/genres/{genre} GET none JSON object with genre description. Example: 
 {
                                genre: 'Action',
                                description: 'Action-packed movies'
                            }
Get data about a director by name /movies/directors/{director} GET none JSON object with data about director. Example: 
{
                                name: 'Bruce Lee',
                                bio: 'Famous martial artist',
                                birthYear: 1940,
                                deathYear: 1973
                            }
Add a new user /users POST JSON object with new user's data. Example: 
{ 
                                name: 'Alex Ribeira',
                                email: 'alex@example.com'
                            }
 JSON object with new user's data, including an ID generated by the system. Example: 
{ 
                                id: 1, 
                                name: 'Alex Ribeira',
                                email: 'alex@example.com'
                            }
Update user information /users/{id} PUT JSON object with updated user data. Example: 
{ 
                                username: 'Shadow_Shinobi' 
                            }
 JSON object with updated user data. Example: 
{
                                id: 1,
                                username: 'Shadow_Shinobi',
                                name: 'Alex Ribeira',
                                email: 'alex@example.com'
                            }
Add a movie to user's favourites /users/{id}/favourites POST JSON object with movie data. Example: 
{ 
                                title: 'Enter the Dragon (1973)' 
                            }
 A text message indicating whether the movie was successfully added: '${movie.title} added to favourites.'
Remove a movie from user's favourites /users/{id}/favourites/{movieTitle} DELETE None A text message indicating whether the movie was successfully removed:'${movie.title} removed from favourites.'
Remove user /users/{id} DELETE None A text message indicating whether the user was successfully removed: 'User with ID: ${userId} successfully removed.'
